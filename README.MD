## How To Use


```bash
# Clone 
$ git clone https://github.com/gamonpo/gofinances.git

# Go into
$ cd gofinances

# Install dependencies
$ yarn install

# Run app
$ expo start
```

## Additional notes

In order to run you need to download the [Expo client](https://expo.dev/client).
